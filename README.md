# This project is Grails Application template.

client side technology is
* grunt.js
* backbone.js
* requirejs
* jquery

## Settings

### install node
http://nodejs.org/

### install grunt

    npm install -g grunt

### init grunt

    cd web-app
    npm init

## Develop

### watch source

    grunt --config grunt-tasks.js

### optimization r.js for release

    grunt --config grunt-tasks.js release
